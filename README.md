# godrecruit

Chrome extension for semi-automated ATS form filling. One-click candidate data extraction.

## Why this exists

Applicant tracking systems make recruiters re-enter the same candidate data across dozens of forms. godrecruit reads a candidate's details once and fills the fields a recruiter would otherwise type by hand, keeping a human in the loop to review before submit.

## What it does

- Extracts candidate data and maps it to ATS form fields.
- Fills supported forms in one click, leaving submit to the user.
- Ships as a browser extension with a Next.js companion surface.

## Stack

TypeScript · Next.js · Chrome Extension APIs
